# Unix Utils (Course)
This was a mini-project for Operating Systems (CSE531) course at IIIT
Hyderabad (Monsoon-2016).

The unix utilities `tr`, `wc`, and a `split` and `tac` hybrid are
implemented in C.

## Build
```
make
```

## Run
For tr
```
./mytr [OPTION]... SET1 [SET2]
```

For wc
```
./mywc [OPTION]... [FILE]...
```

For spltac
```
spltac SRC DEST LINENO
```
